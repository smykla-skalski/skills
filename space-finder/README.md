# Space Finder

Find and switch to macOS Spaces/Desktops by application name.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/space-finder/
```

## Usage

```bash
/space-finder <app-name> [--list] [--current] [--go] [--json]
```

## Skills

- **space-finder**: Find which macOS Space contains a specific app and navigate to it
