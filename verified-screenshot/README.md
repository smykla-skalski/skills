# Verified Screenshot

Capture macOS window screenshots with automatic verification and retry logic.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/verified-screenshot/
```

## Usage

```bash
/verified-screenshot <command> <app> [--verify <strategy>] [--retries N] [--json]
```

## Skills

- **verified-screenshot**: Capture and verify macOS window screenshots with retry logic
