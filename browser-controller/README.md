# Browser Controller

Programmatic control of Chrome and Firefox browsers via CDP and Marionette for tab management, navigation, DOM interaction, and screenshots.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/browser-controller/
```

## Usage

```bash
/browser-controller [command] [args]
```

## Skills

- **browser-controller**: Connect to running browser instances for tab management, navigation, DOM interaction, form filling, JavaScript execution, and screenshots
