# Web Automation

Investigate and implement web browser automation for testing, scraping, and interaction workflows.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/web-automation/
```

## Usage

```bash
/web-automation <url-or-task> [--tool playwright|selenium|puppeteer]
```
