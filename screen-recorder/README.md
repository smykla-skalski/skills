# Screen Recorder

Record macOS screen with verification, retry logic, and format conversion for Discord, GitHub, and JetBrains.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/screen-recorder/
```

## Usage

```bash
/screen-recorder [command] [args]
```

## Skills

- **screen-recorder**: Capture screen recordings of windows or regions with automatic verification and platform-optimized format conversion
