# AI Daily Digest

Daily AI news digest covering technical advances, business news, and engineering impact.

## Installation

```bash
claude --plugin-dir /path/to/sai/ai-daily-digest
```

## Usage

```bash
/ai-daily-digest [--focus technical|business|engineering|leadership] [--notion-page-id ID] [--no-notion]
```
