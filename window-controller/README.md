# Window Controller

Find, activate, and screenshot macOS windows across Spaces.

## Installation

### From GitHub Marketplace

Install from the [SAI plugin collection](https://github.com/smykla-skalski/sai).

### Manual

```bash
claude --plugin-dir /path/to/sai/window-controller/
```

## Usage

```bash
/window-controller <command> <app> [--title <pattern>] [--args-contains <str>] [--json]
```

## Skills

- **window-controller**: Find, activate, and screenshot macOS windows with filtering by app name, title, process path, or command line
